- **What does the man command do? Type in man rm. How do you scroll and get out? 
The man command opens a help document for whatever other command is passed as an argument (you can also do 'man man' to get the man pages on the man command itself). It uses a pager called less, and the way you scroll up/down is using d/u.
- Look at the man page for ls. What does the -l flag do? What does the
- ***a*** flag -do? 
The l flag lists additional information (r/w permissions, last time modified). The a flag shows hidden files
- How do you jump between words in the terminal? 
ctrl + l/r arrow
- How do you get to the end of a line in terminal? 
ctrl + e
- How do you move your cursor to the beginning in terminal? 
ctrl + a

How do you delete a word (without pressing backspace multiple times) in -terminal? 
ctrl + w to delete backward

- What is the difference between a terminal and shell? 
the terminal is the graphical program that displays text/receives input. the shell is the main program that runs inside a terminal, requesting input from the user - and has built-in commands/features for controlling the computer.
- What is an absolute path? 
An absolute path is a path that starts from the root route and gives the complete location for a file or folder
- What is an relative path? 
A relative path is a path that starts at the current location and not the root route
- What is a flag? Give three examples of flags you have used. 
A flag is like a parameter to a command, it slightly modifies the behavior of the command.  Grep -irl (case insentive, recursive, show line)
- What do the r and f flags do with the rm command? 
r removes recursively. f forces removal.
